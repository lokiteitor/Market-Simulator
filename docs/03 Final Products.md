# 1. Objetivo

Este documento define todos los productos finales del sistema económico.

Los productos finales representan:

- bienes consumidos por ciudades
- bienes de uso directo de empresas
- vehículos y maquinaria
- infraestructura avanzada
- tecnología industrial

Estos productos no son transformados posteriormente dentro de cadenas industriales del juego (salvo mantenimiento o consumo operativo definido en otros sistemas).

---

# 2. Filosofía

Los productos finales son el punto de convergencia de toda la economía.

Cada uno:

- depende de múltiples cadenas industriales
- requiere logística compleja
- tiene alta demanda en el mercado global
- genera valor económico significativo

# 4. Relación con la economía

Los productos finales son:

- los principales generadores de ingresos
- los principales consumidores de componentes
- los principales impulsores de demanda industrial

---

## Ejemplo de cadena completa

```
Hierro
→ Acero
→ Chasis
→ Camión
→ Transporte de mercancías
→ Abastecimiento de ciudad
→ Crecimiento económico
→ Mayor demanda de acero
```

---

# 5. Reglas de diseño

- Todo producto final debe consumir múltiples componentes industriales.
- Todo producto final debe depender de al menos 3 cadenas industriales distintas.
- Ningún producto final puede fabricarse directamente desde recursos naturales.
- Los productos finales deben generar demanda constante de logística.
- Cada producto final debe interactuar con el sistema económico global.

---

# 6. Subproductos (integración)

Algunas industrias asociadas a productos finales pueden generar subproductos definidos en el documento 02.

Estos subproductos deben ser:

- reutilizables en otras cadenas
- vendibles en el mercado global
- relevantes económicamente

---

# 7. Cierre del sistema productivo

Con este documento se completa el núcleo de producción del juego:

```
Recursos Naturales
→ Materiales Básicos
→ Materiales Industriales
→ Componentes
→ Productos Finales
→ Consumo / Economía global
```

Este sistema es la base de toda la simulación económica del mundo persistente.
